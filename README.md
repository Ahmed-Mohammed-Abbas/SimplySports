# SimplySports
A plugin to show sport fixtures and game results for enigma2 satellites. 

Installation telnet: cd /usr/lib/enigma2/python/Plugins/Extensions && rm -rf SimplySports && wget --no-check-certificate https://github.com/.../Simply.../archive/refs/heads/main.zip -O SimplySports.zip && unzip SimplySports.zip && mv SimplySports-main SimplySports && rm SimplySports.zip && killall -9 enigma2

New update (V1.1): The new version has a Premier League theme and allows the user to custom-select their leagues.
